# Для монтажёра — 7 рилсов «кейсы методолога»

## Что в пакете
- `teleprompter.md` — дословные тексты, которые Даниил читает в кадре (7 роликов по 18–25 сек)
- `interstitials/` — 21 перебивка, мультяшные 9:16 лупы по 5 сек, единый стиль (бумажный коллаж, крем + чёрный контур + красный)
- Сырые видео с Даниилом он пришлёт отдельно

## Стиль
- Формат: 9:16, 1080×1920
- Шрифт субтитров: **Inter Tight** (жирный), цвет чёрный #111110, выделения красным **#E63311**, подложка не нужна (фон светлый) — субтитры обязательны, рилсы смотрят без звука
- Ключевые цифры можно выносить крупным кеглем на весь экран в момент произнесения
- Перебивки вставлять НА аудио (речь продолжается под ними), длительность врезки 1,5–2,5 сек
- Тумбнейл/постер каждого ролика — первый кадр с Даниилом
